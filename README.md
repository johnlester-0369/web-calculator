# Web Calculator

A clean, keyboard-friendly calculator built with vanilla HTML, CSS, and JavaScript — no build tools required.

## Usage

Open `index.html` in any browser. That's it.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Digits |
| `+` `-` `*` `/` | Operators |
| `.` | Decimal |
| `Enter` or `=` | Equals |
| `Backspace` | Delete last digit |
| `Escape` | Clear |

## Project Structure

```
web-calculator/

└── styles/
```
